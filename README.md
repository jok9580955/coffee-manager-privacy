# 隐私政策 (Privacy Policy)

**更新日期：** 2026年04月08日

欢迎使用“咖啡管家” (以下简称“本应用”)。我们非常重视用户的隐私保护，本隐私政策将帮助您了解我们如何收集、使用、存储您的个人信息。

### 1. 信息收集与使用
我们**不会**在后台收集、存储或上传您的任何个人可识别信息。本应用作为一款单机/本地运行为主的管理工具软件（包括销售记录、账目核算、常客故事记录等功能），您在应用中创建的所有数据将完全保存在您的设备本地。我们未集成任何第三方数据收集、用户行为追踪SDK。

### 2. 权限说明
为了提供相关的服务，我们会向您请求以下权限，我们在使用时均会在本地进行处理，绝不会擅自上传您的隐私数据：
- **相册/相机权限**：用于您可以拍摄或上传照片存入本地“常客故事”或相关数据内。

### 3. 数据存储和安全
您的数据均保存在本地存储结构中（如开启iCloud会在您的个人iCloud中安全同步，受到Apple隐私政策的保护）。我们没有任何自营服务器用于收集或查看您的应用数据，您的数据对我们而言是不可见的。

### 4. 隐私政策的更新
我们可能适时修订本隐私政策，相关隐私页面会在App Store以及该托管页面更新。

### 5. 联系我们
如您对本隐私政策有任何疑问，由于本应用并未收集您的联系方式，您可通过在此页面提交Issue或通过App Store商店页中的支持板块留言来与我们联系。
