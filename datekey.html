隐私政策

最后更新日期: 2026年4月11日

核心承诺: 大特钥匙不收集任何用户数据。
本应用采用纯本地蓝牙技术, 所有操作均在您的设备和车辆之间直接完成, 不经过任何服务器。

1. 数据收集

大特钥匙不收集、不存储、不传输任何个人信息, 包括但不限于:

  • 不收集您的姓名、电话号码、电子邮件地址等个人身份信息
  • 不收集您的位置信息或GPS数据
  • 不收集您的车辆识别号码或车辆信息
  • 不收集您的使用习惯或行为数据
  • 不使用任何第三方分析或广告SDK

2. 蓝牙权限

本应用需要使用蓝牙权限来与您的特斯拉车辆建立本地连接。蓝牙通信仅在您的设备和您的车辆之间进行, 不会将任何蓝牙数据发送到外部服务器。

3. 密钥存储

应用生成的加密密钥仅存储在您设备的安全存储区域(Keychain)中, 采用NIST P-256椭圆曲线加密标准。密钥不会被上传、备份或以任何方式离开您的设备。

4. 网络通信

大特钥匙不需要网络连接即可正常工作。应用不会主动发起任何网络请求, 不连接任何远程服务器。

5. 数据共享

我们不会与任何第三方共享您的数据, 因为我们根本不收集任何数据。

6. 儿童隐私

本应用适用于所有年龄段用户(4+), 不针对儿童收集任何信息。

7. 隐私政策变更

如果我们对隐私政策进行任何更改, 将在此页面上发布更新后的版本并更新日期。

技术支持

常见问题

如何配对蓝牙钥匙

1. 打开大特钥匙应用
2. 确保蓝牙已开启
3. 坐进车内, 在车辆中控屏幕上选择添加钥匙
4. 将NFC卡片放在中控扶手箱上方的读卡区域进行验证
5. 配对完成后即可使用

支持哪些车型

  • Model 3 (所有年款)
  • Model Y (所有年款)
  • Model S (2021年及以后)
  • Model X (2021年及以后)

需要网络连接吗

不需要。大特钥匙完全通过本地蓝牙与车辆通信, 在没有网络的地下车库等环境中也能正常使用。

Apple Watch如何使用

首次使用时需要通过iPhone配对。配对完成后, Apple Watch可以独立使用, 无需iPhone在旁边。

钥匙安全吗

大特钥匙采用与特斯拉官方相同的NIST P-256椭圆曲线加密标准。私钥存储在设备的安全芯片中, 即使设备丢失, 他人也无法提取您的钥匙信息。您也可以随时在车辆中控屏幕上删除已配对的钥匙。
